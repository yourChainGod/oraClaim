一个高效的 ORA 空投批量认领工具，支持多钱包并发处理，具有完善的日志记录和错误处理机制。

## 特性

- 多钱包并发处理（默认10并发）
- 详细的日志记录
- 自动错误重试
- 结果分类记录（成功/失败）
- 支持自定义接收地址

## 使用要求

- Go 1.21+
- 私钥文件（格式：地址----私钥）

```bash

```

3. 按提示输入：
   - 私钥文件路径
